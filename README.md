# PDF Text Finder

Search text in PDF documents with visual highlighting. Upload any PDF, choose a search strategy, and navigate through matches.

## Quick Start

**Prerequisites:** Node.js 18+, Python 3.10+

```bash
# Install dependencies
npm install
pip install -r python/requirements.txt

# Start the app (runs both frontend & backend)
npm run dev
```

Open http://localhost:3000

Or run separately: `npm run dev:next` (frontend) and `npm run dev:python` (backend)

## Tech Stack

**Frontend:** Next.js 15, React 19, TypeScript, PDF.js
**Backend:** FastAPI, pdfplumber, RapidFuzz
**Design:** Dieter Rams-inspired minimal UI
